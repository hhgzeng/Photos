# Personal Image Repository

这是一个个人图片资源仓库，用于存储和管理各类图片资源。

## 目录结构

### 个人相关
- `Avatar/` - 头像图片
- `HomePage/` - 主页相关图片
- `Logo/` - Logo 相关图片

### 图片分类
- `Background/` - 背景图片集合
  - `2D/` - 2D 风格背景
  - `About/` - 关于页面背景
  - `Album/` - 相册背景
  - `Equipment/` - 设备相关图片
  - `Game/` - 游戏相关图片
  - `Music/` - 音乐相关图片
  - `Scenery/` - 风景图片
  - `Video/` - 视频相关图片
  - `Violet/` - 紫罗兰系列图片

### 其他
- `Friend/` - 好友相关图片
- `Payment/` - 支付相关图片
- `YeJiao/` - 夜叫相关图片

## 用途说明

该仓库主要用作个人图床，用于：
1. 博客文章图片存储
2. 个人网站资源引用
3. 各类文档图片托管
4. 社交媒体图片存储

## 注意事项

- 图片按照类别分类存储，便于管理和查找
- 建议使用相对路径引用图片
- 定期维护和清理未使用的图片资源
